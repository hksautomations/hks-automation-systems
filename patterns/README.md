# Automation & AI Design Patterns

This folder documents **reusable design patterns**
used across AI automation systems.

The focus is on:
- Reliability
- Control
- Maintainability
- Safe AI usage

These patterns are **tool-agnostic** and can be applied
across different workflows and platforms.
