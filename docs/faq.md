# Frequently Asked Questions

## Is everything fully automated?
No. High-risk decisions always allow human review.

## Can the system be modified later?
Yes. All systems are modular and extendable.

## What tools are required?
Primarily n8n and Large Language Models, but systems are tool-agnostic.

## Is client data safe?
Yes. Data handling follows least-access and minimal exposure principles.

## Can automation be paused?
Yes. Every system includes manual and automatic kill switches.
